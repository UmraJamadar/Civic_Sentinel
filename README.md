->Civic Sentinel
An AI-powered civic complaint management system that automatically 
detects urban infrastructure issues and routes complaints to the 
correct municipal department.

-->Features
- AI detection of potholes & garbage using YOLOv11m
- Auto complaint generation using Gemini AI
- GPS location tagging
- Smart routing to correct department
- User complaint dashboard
- Municipality department portal

Tech Stack
- AI Model: YOLOv11m (trained on 5032 images)
- AI Description: Google Gemini API
- Backend: FastAPI + Python
- Frontend: React.js
- Database: MongoDB

Built for smart city infrastructure management.
